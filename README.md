# Odoo Clone - Contact Management

Aplicación estilo Odoo desarrollada con Flask, PostgreSQL y desplegada en Render.

## Funcionalidades
- Login y registro de usuarios
- Gestión de contactos (CRUD completo)
- Interfaz similar a Odoo

## Tecnologías
- Python Flask
- PostgreSQL (Render)
- Flask-Login
- HTML/CSS estilo Odoo

## Despliegue en Render
1. Crear base de datos PostgreSQL en Render
2. Conectar repositorio de GitHub
3. Configurar variable de entorno DATABASE_URL
4. Hacer deploy automático
